## 三、关键发现

### 3.1 Web Search V2 显著优势

1. **CPI全面提升**: 平均提升18.3%，最大提升33.9%（患者708387）
2. **PTR大幅提升**: 从0.128提升至0.386（+201.6%），证据可追溯性显著改善
3. **CER大幅降低**: 从0.222降低至0.074（-66.7%），临床错误率显著降低

### 3.2 RAG V2 主要问题

1. **虚假章节引用**: Patient Imaging Report、Patient History等章节不存在于检索文档
2. **Parametric Knowledge过度使用**: 缺乏具体来源支撑
3. **缺乏PubMed引用**: 所有患者均无PubMed或OncoKB引用

### 3.3 Web Search V2 主要优势

1. **丰富的PubMed引用**: 平均6.5处/例，可追溯性强
2. **规范的指南引用**: [Guideline: ...]格式统一
3. **更低的错误率**: CER仅0.074，较RAG V2降低66.7%
