## 评审说明

本模板用于临床专家对BM Agent生成的MDT报告进行人工评审，评估以下三个指标：
- **CCR** (Clinical Consistency Rate): 临床决策一致性率
- **MQR** (MDT Quality Rate): MDT报告规范度
- **CER** (Clinical Error Rate): 临床错误率

评审完成后，请将评分填入 `analysis/reports/clinical_review_results.csv`

---

## 一、CCR (Clinical Consistency Rate) 评审

### 1.1 评分标准

CCR 采用 **5个维度、每项0-4分制**：

**治疗线判定准确性**:
- 4分: 完全准确，与临床实际完全一致
- 3分: 基本准确，存在轻微偏差但不影响治疗决策
- 2分: 部分准确，存在明显偏差
- 1分: 明显错误
- 0分: 完全错误或遗漏

**方案选择合理性**:
- 4分: 推荐方案完全符合NCCN/ASCO指南
- 3分: 推荐方案基本合理，有小瑕疵
- 2分: 部分合理，需调整
- 1分: 不合理，可能有害
- 0分: 严重错误或遗漏

**剂量参数准确性**:
- 4分: 剂量与引用均准确
- 3分: 基本正确，有轻微偏差
- 2分: 范围基本正确但不够精确
- 1分: 剂量明显错误
- 0分: 未提及或严重错误

**排他性论证充分性**:
- 4分: 充分论证，引用可靠
- 3分: 基本充分，略有不足
- 2分: 有论证但不够详细
- 1分: 论证薄弱
- 0分: 未提供论证

**临床路径一致性**:
- 4分: 与真实临床路径完全一致
- 3分: 基本一致，细节有差异
- 2分: 部分一致
- 1分: 明显偏离
- 0分: 完全偏离

## 二、MQR (MDT Quality Rate) 评审

### 2.1 评分标准

**8模块完整性** (40%):
- 2分: 8个模块齐全，无缺失
- 1分: 7个模块完整，1个模块缺失
- 0分: 缺失2个及以上模块

**模块适用性判断** (30%):
- 2分: Module 6 (围手术期管理) 标记N/A合理，其他模块适用
- 1分: 1个模块适用性判断不当
- 0分: 2个及以上模块适用性判断错误

**结构化程度** (20%):
- 2分: 使用表格、列表、分级标准（如RANO-BM）
- 1分: 部分结构化，存在大段文字
- 0分: 缺乏结构化，难以快速获取关键信息

**不确定性处理** (10%):
- 2分: 对缺失数据明确标注，提供替代方案
- 1分: 提及不确定性但处理不够充分
- 0分: 未处理不确定性或忽略重要缺失数据

## 三、CER (Clinical Error Rate) 评审

### 3.1 错误分级标准

**严重错误** (权重3):
- 推荐禁忌治疗方案（如KRAS突变患者推荐EGFR抑制剂）
- 治疗线判定错误导致方案类别错误
- 遗漏危及生命的并发症处理

**主要错误** (权重2):
- 剂量计算错误（>20%偏差）
- 药物相互作用未识别
- 重要禁忌症未识别

**次要错误** (权重1):
- 随访时间间隔不当
- 支持治疗建议不完善
- 引用文献与声称不符

## 四、综合评分计算

### 4.1 单项指标计算

```
CCR, MQR: 每位患者先得到0-4分制总分，再对9例取均值
CER: CER = 加权错误分 / 最大可能错误分，范围为0-1
PTR: PTR = ΣTraceScore / (2N)，范围为0-1
```

### 4.2 CPI (Composite Performance Index) 计算

**公式**:
```
CPI = 0.35 × (CCR/4) + 0.25 × PTR + 0.25 × (MQR/4) + 0.15 × (1 - CER)
```
