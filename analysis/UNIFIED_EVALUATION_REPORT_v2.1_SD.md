## 附录A: 计算方法详解

### A.1 标准差计算

使用**样本标准差**（n=9）：
```
SD = √[Σ(xi - x̄)² / (n-1)]
```

### A.2 置信区间计算

95%置信区间（t分布，df=8）：
```
95% CI = Mean ± t(0.025,8) × SE
       = Mean ± 2.306 × (SD/√9)
       = Mean ± 0.769 × SD
```

### A.3 变异系数（CV）
```
CV = (SD / Mean) × 100%
```
