## Module 2: 个体化治疗方案 (Primary Personalized Plan)

### 2.1 治疗线判定
**二线/后线治疗**
- 依据：患者已接受埃克替尼（第一代 EGFR-TKI）治疗约 2 年半，当前出现疾病进展（新发脑转移）
- 治疗线：二线治疗（EGFR-TKI 耐药后）

### 2.2 局部治疗方案

#### 方案 A：立体定向放射外科 (SRS)
**推荐作为首选局部治疗**

- **适应证**: 患者有症状性脑转移，病灶最大径 16mm（1.6cm），符合 SRS 适应证 [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 2.1]
- **处方剂量**: 
  - 对于 1.6cm 病灶：20-24Gy/1 次分割 [Local: Radiation_Therapy_for_Brain_Metastases.md, Table 3, Recommendation 3]
  - 或 27Gy/3 次分割（如考虑病灶靠近功能区）[Local: Radiation_Therapy_for_Brain_Metastases.md, Table 3, Recommendation 4]
- **靶区**: 左额顶叶两处转移灶（GTV）+ 1-2mm 边缘（PTV）
- **证据等级**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 3.2] [PubMed: PMID 27458945]

#### 方案 B：手术治疗
**备选方案（如 SRS 不可行或需要病理确诊）**

- **适应证**: 症状性、占位效应明显、需要明确病理诊断
- **手术方式**: 左额顶叶转移瘤显微镜下切除术
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 1.1]

### 2.3 系统治疗方案

#### 2.3.1 分子检测指导的靶向治疗（优先推荐）

**第一步：基因检测（紧急）**
- **检测项目**: 
  - EGFR T790M 突变（血浆 ctDNA 或组织活检）
  - EGFR 全外显子测序（18-21 外显子）
  - 耐药机制检测（MET 扩增、HER2 扩增、C797S 突变等）
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 380-400]

**第二步：根据检测结果选择方案**

**情景 A: T790M 阳性**
- **推荐方案**: 奥希替尼（Osimertinib）80mg 口服 每日一次
- **依据**: FLAURA 研究证实奥希替尼 CNS 缓解率>60%，CNS PFS 显著优于一代 TKI
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 350-380] [PubMed: PMID 37691552]

**情景 B: T790M 阴性**
- **推荐方案**: 含铂双药化疗 ± 贝伐珠单抗
  - 培美曲塞 500mg/m² + 卡铂 AUC 5，每 3 周一次，4-6 周期
  - 或 培美曲塞 500mg/m² + 顺铂 75mg/m²，每 3 周一次，4-6 周期
- **维持治疗**: 培美曲塞 500mg/m² 单药维持（如一线未使用）
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 400-420] [PubMed: PMID 37920971]

#### 2.3.2 免疫联合治疗（备选）
- **方案**: 阿替利珠单抗 + 贝伐珠单抗 + 紫杉醇 + 卡铂（IMpower150 方案）
- **适应证**: TKI 和化疗失败后
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 430-450]

### 2.4 推荐治疗顺序
1. **紧急处理**: 地塞米松减轻脑水肿（见 Module 3）
2. **局部治疗**: SRS（优先）或手术（如占位效应严重）
3. **基因检测**: 同步进行（血浆 ctDNA 优先，如阴性则组织活检）
4. **系统治疗**: 根据基因检测结果选择奥希替尼或化疗

## Module 3: 支持治疗 (Systemic Management)

### 3.1 脑水肿管理
- **地塞米松**: 4-8mg 静脉注射/口服 每 6-8 小时一次
  - 根据症状调整剂量，症状缓解后逐渐减量
  - 预防性使用质子泵抑制剂（如奥美拉唑 20mg QD）预防消化道溃疡
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Clinical Interpretation Section]

### 3.2 癫痫管理
- **病史**: 患者 2 年半前有癫痫大发作史
- **预防用药**: 如当前无癫痫发作，不推荐预防性使用抗癫痫药物
- **治疗用药**: 如出现癫痫发作，首选左乙拉西坦 500-1000mg BID
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Guideline Text]

### 3.3 止吐治疗
- **适应症**: 患者有呕吐症状
- **方案**: 昂丹司琼 8mg 静脉注射/口服 每 8 小时一次（按需）

### 3.4 营养支持
- 评估营养状态，必要时给予肠内营养支持

### 4.1 影像学随访
- **头颅 MRI 增强**: 
  - 局部治疗后 2-3 个月复查
  - 之后每 2-3 个月一次，持续 1-2 年
  - 稳定后每 4-6 个月一次
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Implementation Remark] [Local: Radiation_Therapy_for_Brain_Metastases.md, Text]

### 4.2 疗效评估标准
- **脑转移瘤**: RANO-BM 标准
- **颅外病灶**: RECIST 1.1 标准

### 4.3 神经系统评估
- 每次随访评估神经系统症状、KPS/ECOG 评分

### 4.4 不良反应监测
- **SRS 后**: 监测放射性坏死（头痛、癫痫、局灶神经功能缺损）
- **奥希替尼**: 监测 QT 间期延长、间质性肺炎、皮疹、腹泻
- **化疗**: 监测血常规、肝肾功能、骨髓抑制

## Module 5: 被排除方案 (Rejected Alternatives)

### 5.1 继续埃克替尼治疗
- **排除理由**: 患者已出现明确疾病进展（新发脑转移），继续原方案无效
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 380-400]

### 5.2 单纯全脑放疗 (WBRT)  alone
- **排除理由**: 
  - 患者病灶数量有限（2 处），SRS 局部控制率更高且神经认知毒性更低
  - WBRT 神经认知毒性显著，仅在多发病灶（>10 个）或 SRS 不可行时考虑
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 3.2] [Local: Radiation_Therapy_for_Brain_Metastases.md, kq1]

### 5.3 单纯系统治疗（延迟局部治疗）
- **排除理由**: 患者有症状（言语不清、头痛、呕吐），根据 ASCO/SNO/ASTRO 指南，症状性脑转移应接受局部治疗，不应延迟
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 2.1]

### 5.4 免疫单药治疗
- **排除理由**: EGFR 突变 NSCLC 对免疫单药响应率低（ORR 约 12%），且患者有症状需要快速缓解
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 430-450]

## Module 6: 围手术期/放疗期管理 (Peri-procedural Holding Parameters)

### 6.1 抗血管生成药物管理
- **贝伐珠单抗**（如使用）: 
  - 手术前至少停药 6 周
  - 手术后至少停药 4 周，待伤口完全愈合
- **证据**: [PubMed: PMID 28687377]

### 6.2 EGFR-TKI 管理
- **埃克替尼/奥希替尼**:
  - SRS 前：建议停药 3-5 天（减少放射性坏死风险，尽管证据有限）
  - SRS 后：可在放疗后 3-5 天恢复使用
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Text]（专家意见）

### 6.3 化疗药物管理
- **培美曲塞/铂类**:
  - SRS 前后各停药 1 周
  - 如同步放化疗，需密切监测骨髓抑制
- **证据**: [PubMed: PMID 28687377]

### 6.4 糖皮质激素管理
- **地塞米松**: SRS 当天及之后 3-5 天维持或略增剂量，之后逐渐减量
- **证据**: [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Clinical Interpretation]

### 6.5 抗凝药物管理
- 患者无抗凝药物使用史，无需特殊管理

## Module 7: 分子病理学检测建议 (Molecular Pathology Orders)

### 7.1 优先检测项目（紧急）

#### EGFR T790M 突变检测
- **样本类型**: 血浆 ctDNA（优先）或 组织活检
- **检测方法**: NGS 或 ARMS-PCR
- **临床意义**: 一代 EGFR-TKI 耐药后，约 50-60% 患者出现 T790M 突变，阳性者可使用奥希替尼
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 380-400] [Local: ASCO_SNO_ASTRO_Brain_Metastases.md, Recommendation 2.3]

#### EGFR 全外显子测序（18-21 外显子）
- **检测内容**: 18、19、20、21 外显子全序列
- **临床意义**: 明确原发突变类型（19del vs L858R），检测罕见耐药突变（如 C797S、L718Q 等）
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 100-150]

### 7.2 扩展检测项目（如条件允许）

#### 耐药机制多基因 panel
- **检测基因**: MET、HER2、KRAS、BRAF、PIK3CA 等
- **临床意义**: 检测旁路激活（如 MET 扩增）或其他耐药机制
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 400-420]

#### PD-L1 表达检测
- **检测方法**: IHC（22C3 抗体）
- **临床意义**: 如后续考虑免疫联合治疗，需评估 PD-L1 表达水平
- **证据**: [Local: Oncogene-addicted_metastatic_non-small-cell_lung_cancer_ESMO_Clinical_Practice_Guideline_for_diagnosis,_treatment_and_follow-up.md, Line 约 430-450]

### 7.3 检测时间线
- **血浆 ctDNA**: 立即送检（3-5 个工作日）
- **组织活检**: 如血浆阴性且临床需要，考虑再次活检（左额顶叶病灶或颅外病灶）

## 总结与推荐

### 核心推荐
1. **局部治疗**: 立体定向放射外科（SRS）20-24Gy/1f 或 27Gy/3f（优先）
