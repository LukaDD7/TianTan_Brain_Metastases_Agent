## V3.0 关键规则变更

### PTR 物理可追溯率 V3.0

**双源溯源机制**:
- **Tier 1A** (外部权威循证): PubMed/OncoKB/指南章节，权重 **1.0**
- **Tier 1B** (内部患者事实): 患者输入（年龄、既往史、症状），权重 **1.0** ⭐**V3.0新增**
- **Tier 2** (泛指南引用): 权重 **0.3**
- **Tier 3** (白名单Web): .gov/.edu/药企官网，权重 **0.1**
- **Tier 3C** (商业网站): PTR记 **0分**，CER追加 **+2.0** Major Error

**安全拒答处理**:
- "未在检索中明确"、"需临床医师决定"等声明
- **分母计入**，分子记 **0分**（不剔除，真实反映硬核证据覆盖率）

**计算公式**:
```
PTR = (Tier 1A + Tier 1B + Tier 2 + Tier 3) / 总声明数
```

**CPI计算公式** (不变):
```
CPI = 0.35×(CCR/4) + 0.25×PTR + 0.25×(MQR/4) + 0.15×(1-CER)
```
