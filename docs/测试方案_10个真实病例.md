## 三、真实病例测试方案（10 个案例）

### 3.1 方案选择

#### **方案 A：直接解析长表格式（快速，但代码复杂）**
- **优点：** 无需预处理，直接读取原始数据
- **缺点：** 每次测试需要解析长表，代码逻辑较复杂

#### **方案 B：转换为宽表格式（推荐，代码简洁）**
- **优点：**
  - 代码简单清晰（每行 = 1 个病例）
  - 便于人工检查和修改测试数据
  - 可以添加预期结果列（用于断言验证）
- **缺点：** 需要预处理步骤

### 3.2 推荐方案：宽表格式转换

#### **步骤 1：运行预处理脚本**
```bash
python scripts/prepare_test_cases.py
```

**脚本内容（`scripts/prepare_test_cases.py`）：**
```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
"""
将长表格式测试数据转换为宽表格式
"""

import pandas as pd
from pathlib import Path

def convert_long_to_wide(excel_path: str, output_path: str):
    """将长表格式转换为宽表格式"""
    df = pd.read_excel(excel_path)

    print(f"读取数据：{len(df)} 行")
    print(f"列：{list(df.columns)}")

    # 透视表：id 为索引，Feature 为列，Value 为值
    wide_df = df.pivot(index='id', columns='Feature', values='Value')

    # 重置索引，使 id 变为普通列
    wide_df = wide_df.reset_index()

    # 保存为新的 Excel
    wide_df.to_excel(output_path, index=False)

    print(f"\n✅ 转换完成：{output_path}")
    print(f"包含 {len(wide_df)} 个病例")
    print(f"字段（{len(wide_df.columns)} 个）：")
    for col in wide_df.columns:
        print(f"  - {col}")

    # 显示前 3 行
    print(f"\n前 3 行数据：")
    print(wide_df.head(3).to_string())

    return wide_df

if __name__ == "__main__":
    PROJECT_ROOT = Path(__file__).parent.parent
    INPUT_PATH = PROJECT_ROOT / "workspace" / "original_cases" / "TT_BM_test.xlsx"
    OUTPUT_PATH = PROJECT_ROOT / "workspace" / "test_cases_wide.xlsx"

    convert_long_to_wide(INPUT_PATH, OUTPUT_PATH)
```

### 3.3 基因突变提取策略

由于真实病历中**不一定明确写出基因突变**，我们需要基于癌症类型推断常见突变：

```python
def extract_mutation_info_from_case(case_data: Dict[str, Any]) -> Optional[Dict[str, str]]:
    """
    从病历中提取基因突变信息

    策略：
    1. 优先从现病史/诊断中查找明确的基因名称（EGFR、BRAF、ALK 等）
    2. 如果没有明确写出，基于癌症类型推断常见突变

    返回：
    - {"gene": "BRAF", "variant": "V600E"} 如果检测到突变
    - None 如果无突变或无法确定
    """
    # 提取文本
    text = ""
    for key in ['现病史', '出院诊断', '入院诊断', '病理诊断']:
        if key in case_data and pd.notna(case_data[key]):
            text += str(case_data[key]) + " "

    # 规则 1: 查找明确的基因名称
    gene_patterns = {
        'EGFR': ['EGFR', '表皮生长因子受体'],
        'BRAF': ['BRAF', 'BRAF V600E', 'BRAF V600K'],
        'ALK': ['ALK', '间变性淋巴瘤激酶'],
        'ROS1': ['ROS1'],
        'KRAS': ['KRAS', 'KRAS G12C'],
        'NRAS': ['NRAS'],
        'KIT': ['KIT', 'c-KIT'],
        'PDGFRA': ['PDGFRA']
    }

    for gene, patterns in gene_patterns.items():
        for pattern in patterns:
            if pattern in text:
                # 尝试提取具体变异
                variant = extract_variant_from_text(text, gene)
                return {"gene": gene, "variant": variant or "Unknown"}

    # 规则 2: 基于癌症类型推断
    diagnosis = case_data.get('出院诊断', '') + case_data.get('入院诊断', '')

    if '黑色素瘤' in diagnosis or 'melanoma' in diagnosis.lower():
        return {"gene": "BRAF", "variant": "V600E"}  # 黑色素瘤常见 BRAF V600E
    elif '肺癌' in diagnosis or 'NSCLC' in diagnosis or '腺癌' in diagnosis:
        return {"gene": "EGFR", "variant": "L858R"}  # 肺腺癌常见 EGFR 突变
    elif '胃肠道间质瘤' in diagnosis or 'GIST' in diagnosis:
        return {"gene": "KIT", "variant": "exon11"}  # GIST 常见 KIT 突变
    else:
        return None
```
