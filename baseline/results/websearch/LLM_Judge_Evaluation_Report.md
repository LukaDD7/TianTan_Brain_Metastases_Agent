## 三、核查的错误细节汇总

### 3.1 WebSearch严重错误

**错误1: 治疗线判定与方案选择矛盾 (患者868183)**
```
- 判定: "二线治疗" ✓
- 推荐: "信迪利单抗+培美曲塞+卡铂" (一线方案) ✗
- 错误性质: 临床决策矛盾
- 潜在危害: 重复已耐药方案，延误二线治疗
- 正确方案: 多西他赛 75mg/m² (二线标准)
- BM Agent处理: ✅ 正确推荐多西他赛+PMID验证
```

**错误2: 引用不可追溯**
```
- 现象: 使用[1,45], [34,58]等数字引用
- 问题: 无法验证来源真实性
- 风险: LLM可能编造引用
- BM Agent处理: ✅ PMID可独立验证
```

**错误3: 分子证据无验证**
```
- 现象: "STK11...免疫微环境冷肿瘤特征"
- 问题: 无OncoKB API验证
- 实际: STK11 G279Cfs*6为Likely Oncogenic, LEVEL_4 (Bemcentinib+Pembrolizumab)
- 风险: 可能误导免疫治疗决策
- BM Agent处理: ✅ OncoKB API实时查询
```

**错误4: Module 6机械填充**
```
- 现象: 患者无手术计划，但详细描述围手术期管理
- 问题: 缺乏模块适用性判断
- BM Agent处理: ✅ 正确标记N/A
```
